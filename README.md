## 기능

1. 로그인, 비밀번호 관리
2. 학사정보 조회 및 수정
3. 강의 조희 및 수강 신청, 수정
4. 학사일정 조회
5. 성적 조회, 출력
6. 교수 조회
7. 등록금 조회 및 출력
8. 게시판 등록, 수정, 삭제, 조회
9. 스터디 신청, 등록, 수정, 삭제
10. 관리자 기능
